The **GenAI Voice of Customer (VoC) System** is an AI-powered solution that automates customer feedback analysis using Generative AI. 
It can understand multi-topic messages, detect mixed sentiments, and generate personalized responses in real time. 
The system routes each issue to the correct department, helping businesses reduce response time and improve customer support. 
Built using Google Bard, OpenAI GPT models, and standard NLP techniques, this project streamlines feedback handling 
by combining message classification, sentiment detection, and automated response generation into one efficient workflow.
